# 音乐制作人 - 日程规划助手

一个专为音乐制作人设计的日程规划和进度跟踪工具。

## 功能特点

- 🎵 **歌曲管理**：添加、编辑、删除歌曲，跟踪每首歌的制作进度
- 📊 **进度跟踪**：实时显示任务完成情况和整体进度
- ⏱️ **时间管理**：跟踪每首歌的预计时长和已用时长
- 📅 **时间线规划**：自动计算每天需要的工作时长，合理分配时间
- ✅ **任务清单**：10个标准制作流程任务，确保不遗漏任何步骤
- 📈 **统计信息**：一目了然的完成情况统计

## 制作流程

每首歌的制作包含以下10个任务：

1. 听10张专辑，确定子曲风
2. 找到3首参考歌，发给队长
3. 制作Demo，自己觉得OK
4. Demo发给队长，获取意见
5. 完成编曲
6. 编曲发给队长，获取意见
7. 编曲OK，开始混音母带
8. 混音母带发给队长，获取意见
9. 队长OK，发给校长
10. 校长OK，完成制作

## 时间分配

- **编曲作曲**：70% 的时间
- **混音母带**：30% 的时间
- **每首歌预计时长**：20-30 小时（可自定义）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"添加新歌"按钮创建第一首歌
3. 填写歌曲信息，包括：
   - 歌曲名称
   - 子曲风
   - 预计时长（默认25小时）
   - 当前阶段
   - 已完成的任务（勾选复选框）
   - 已用时长
   - 备注
4. 保存后即可在列表中查看和管理
5. 系统会自动计算：
   - 每首歌的完成进度
   - 剩余工作时间
   - 每天建议工作时长
   - 整体项目进度

## 数据存储

所有数据存储在浏览器的 localStorage 中，无需服务器，完全本地运行。

## 目标设置

- **总天数**：180 天
- **剩余天数**：175 天
- **目标歌曲数**：9 首

## 技术栈

- HTML5
- CSS3 (现代渐变设计)
- 原生 JavaScript (无依赖)

## 浏览器支持

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
